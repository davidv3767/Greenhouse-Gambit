# Greenhouse-Gambit
This is an escape room made by David in order to practice Python.

Ever feel bored of the escape rooms you have to do in class or the ones that have a really good story, but you can only play it once? Introducing Greenhouse Gambit... an escape room based on randomization and a story. Please note this only runs on Windows (if reviewing for Flavortown, please see Devlog #14 or #15).

How To Play:
Press 1 to start game
Then follow instructions to solve the puzzles. Sometimes, you may get lucky if you try choices 2 or 3, but it makes the process less fun.

# Greenhouse-Gambit
This is an escape room made by David in order to practice Python.

Ever feel bored of the escape rooms you have to do in class or the ones that have a really good story, but you can only play it once? Introducing Greenhouse Gambit... an escape room based on randomization and a story. Please note this only runs on Windows (screenshot of demo is Devlog #15 for flavortown).

How To Play:
Press 1 to start game
Then follow instructions to solve the puzzles. Sometimes, you may get lucky if you try choices 2 or 3, but it makes the process less fun.

Answers or Hints:
Due to the hardness of the room (very hard, according to testers), I realize some people may not be able to play the room properly. This makes it WAY less fun (enjoying the challenge is the goal of the room), but anyways, here are the answers (meant for Flavortown testers):

Guessing The Number w/ 6 Hints --> Go ahead and use AI (it's too randomized for me to accurately help you).

Encrypted Sentence -- The PINs to try are 3739, 2486, 4985.

Riddle --> The key is calculus.

Colors --> Note these equations: purple + blue = green, red + yellow = orange, blue + blue = orange, red + red = green, & yellow + blue = green.

Love Letter --> The message is "flavortown is the best".

Morse Code --> Use a morse translator or try these: spring, word, morse, hacking.

Guessing The Number w/ 1 Hint (Size) --> I don't know a hint that can help, sadly. Just hope you're lucky...

Multi-Step Encryption --> Try randomization, exposition, or rising.

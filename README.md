Short Summary: Ever feel bored of the escape rooms you have to do in class or the ones that have a really good story, but you can only play it once? Introducing Greenhouse Gambit... an escape room based on randomization, replayability, and a small story!

Disclaimer: Please note this only runs on Windows (as per my knowledge; I don't have the time to make it work on other operating systems).

How To Play: Press 1 to start the game. Then, follow the console-printed instructions to solve the puzzles. Sometimes, you may get lucky if you try choices 2 or 3, but it makes the gameplay less fun overall.
